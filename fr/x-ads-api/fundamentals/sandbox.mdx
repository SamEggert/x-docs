---
title: Sandbox
---

Le sandbox est un environnement qui vous permet de tester et de développer votre implémentation de l’Ads API.

Les campagnes créées via le sandbox ne seront pas diffusées aux utilisateurs et des enregistrements de test pour des objets tels que les Funding Instruments peuvent être créés dynamiquement.

Utiliser le sandbox consiste simplement à appeler le nom d’hôte approprié ads-api-sandbox.x.com, mais il contient également des fonctionnalités spécifiques propres au sandbox.

Envoyez d’abord une requête à [POST accounts](/fr/x-ads-api/campaign-management#post-accounts) pour créer un nouveau compte annonceur. La réponse à cet appel contiendra un account_id qui pourra être utilisé pour les requêtes suivantes. Ces endpoints permettent de créer et de tester plusieurs comptes annonceurs.

<div id="sandbox-only-endpoint-list">
  ### Liste des endpoints réservés au sandbox :
</div>

* [POST accounts](/fr/x-ads-api/campaign-management#post-accounts) Créer un compte dans le sandbox
* [POST accounts/:account_id/features](/fr/x-ads-api/campaign-management#post-accounts-account-id-features) Ajouter une fonctionnalité de compte dans le sandbox
* [POST accounts/:account_id/funding_instruments](/fr/x-ads-api/campaign-management#post-accounts-account-id-funding-instruments) Ajouter un instrument de financement à un compte dans le sandbox
* [DELETE accounts/:account_id](/fr/x-ads-api/campaign-management#delete-accounts-account-id) Supprimer un compte dans le sandbox
* [DELETE accounts/:account_id/features](/fr/x-ads-api/campaign-management#delete-accounts-account-id-features) Supprimer une fonctionnalité de compte d’un compte dans le sandbox
* [DELETE accounts/:account_id/funding_instruments/:funding_instrument_id](/fr/x-ads-api/campaign-management#delete-accounts-account-id-funding-instruments-funding-instrument-id) Supprimer un instrument de financement d’un compte dans le sandbox
