---
title: Limites de débit
icon: gauge-simple-high
---

Chaque jour, des milliers de développeurs envoient des requêtes vers la plateforme développeur X. Pour nous aider à gérer le volume considérable de ces requêtes, des limites sont imposées au nombre de requêtes pouvant être effectuées. Ces limites nous permettent de proposer une API fiable et évolutive sur laquelle notre communauté de développeurs s’appuie.

Chacune de nos API utilise les limites de débit différemment. Pour en savoir plus sur ces différences entre plateformes, veuillez consulter les pages dédiées aux limites de débit dans les sections de nos API respectives :

* [X API v2](/fr/x-api/fundamentals/rate-limits)
* [X API: Enterprise](/fr/x-api/enterprise-gnip-2.0/fundamentals/rate-limits)
* [X Ads API](/fr/x-ads-api/fundamentals/rate-limiting)
