  ## Comparaison des endpoints de masquage des réponses de l’API X
</div>

L’endpoint v2 de masquage des réponses remplace l’endpoint Labs de masquage des réponses. Si vous avez du code, des applications ou des outils qui utilisent la version Labs de cet endpoint et que vous envisagez de migrer vers le nouvel endpoint X API v2, ce guide est pour vous.

Pour utiliser la nouvelle X API v2 (y compris l’endpoint de masquage des réponses), vous devrez [vous inscrire au nouveau portail développeur](https://developer.x.com/en/portal/opt-in), créer un [Project](/fr/resources/fundamentals/projects) et ajouter une App à ce Project. Vous pourrez ensuite utiliser les identifiants associés à cette App pour effectuer des requêtes vers l’endpoint de masquage des réponses. Ajouter la même App inscrite pour l’endpoint Labs v2 de masquage des réponses maintiendra l’authentification de vos utilisateurs.

Le tableau suivant compare les différences entre Labs et le nouvel endpoint X API v2 :
 

| **Description** | **Labs v2** | **X API v2** |
| :--- | :--- | :--- |
| Host domain | https://api.x.com | https://api.x.com |
| Endpoint path | /labs/2/tweets/:id/hidden | /2/tweets/:id/hidden |
| Authentication | OAuth 1.0a User context | OAuth 1.0a User context |
| HTTP methods supported | PUT | PUT |
| Default request rate limits | 10 requêtes par 15 minutes (partagées entre tous les utilisateurs authentifiés) | 50 requêtes par 15 minutes (par utilisateur authentifié) |
| Can hide replies | ✔︎  | ✔︎  |
| Can unhide a previously hidden reply | ✔︎  | ✔︎  |
| Can hide or unhide replies multiple times | ✔︎  | ✔︎  |
| Requires the use of credentials from a [developer App](/fr/resources/fundamentals/developer-apps) associated with a [project](/fr/resources/fundamentals/projects) |     | ✔   |
