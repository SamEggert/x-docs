  ## Applications utilisant l’endpoint de masquage des réponses
</div>

Voici une sélection d’applications remarquables créées par des développeurs ayant intégré l’endpoint de masquage des réponses. En combinant leur technologie avec l’API X, ils ont conçu des expériences utiles pour aider tous les utilisateurs à améliorer la conversation publique.

![Clarabridge](https://cdn.cms-twdigitalassets.com/content/dam/developer-twitter/hide-replies/clarabridge.jpg.twimg.1920.jpg)

**Clarabridge**

Clarabridge intègre le masquage des réponses afin que les entreprises puissent garder leur fil X propre et agréable à lire. Les réponses peuvent être masquées manuellement, et les agents peuvent aussi utiliser l’analyse de texte de Clarabridge pour détecter automatiquement les grossièretés et autres propos non conformes à la politique de la marque.

[Aller sur le site ▸](https://www.clarabridge.com/ "Aller sur le site ▸")

[Clarabridge sur X](https://x.com/Clarabridge "Clarabridge sur X")

![Perspective API template app](https://cdn.cms-twdigitalassets.com/content/dam/developer-twitter/hide-replies/perspective-logo.png.twimg.1920.png)

**Application modèle Perspective API**

Perspective est une intelligence artificielle entraînée par Jigsaw (une entité d’Alphabet) pour détecter les commentaires toxiques. Cette application détecte les réponses entrantes d’une personne et masque automatiquement une réponse en fonction du « score » indiquant le degré de confiance de Perspective qu’un commentaire ressemble à des commentaires toxiques déjà observés. Il s’agit d’une application modèle open source développée par X. Nous vous encourageons à la personnaliser pour créer des outils pour vos utilisateurs.

[Utiliser l’application ▸](https://quintessential-yoke.glitch.me "Utiliser l’application ▸")

[Jigsaw sur X](https://x.com/jigsaw "Jigsaw sur X")

![Reshuffle](data:image/svg+xml,%3Csvg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 150 150%22%3E%3C/svg%3E)

**Reshuffle**

Reshuffle est une plateforme permettant de connecter des applications métier à l’aide de scripts flexibles. Ils ont créé un script d’intégration pour détecter et masquer les réponses contenant les mots-clés que vous définissez. Pour l’essayer, assurez-vous de disposer d’une application valide avec la fonction Masquer les réponses activée, puis suivez les instructions du dépôt.

[Aller à l’application ▸](https://github.com/reshufflehq/reshuffle "Aller à l’application ▸")

[Reshuffle sur X](https://x.com/reshufflehq "Reshuffle sur X")

![Masquer les réponses indésirables](data:image/svg+xml,%3Csvg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 396 397%22%3E%3C/svg%3E)

**Masquer les réponses indésirables**

Dara Oladosu, le développeur derrière l’application populaire QuotedReplies, a créé une application qui masque automatiquement les réponses répondant à certains critères qu’il a établis comme étant plus susceptibles d’indiquer un comportement abusif, notamment les réponses contenant certains mots-clés qu’il a déjà mis en sourdine.

[Aller à l’application ▸](https://hideunwantedreplies.com/ "Aller à l’application ▸")

[Dara Oladosu sur X](https://x.com/dara_tobi "Dara Oladosu sur X")

![Pandaflow](data:image/svg+xml,%3Csvg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 94 94%22%3E%3C/svg%3E)

**Pandaflow**

Pandaflow est une plateforme no code pour automatiser les workflows. Ils ont intégré l’endpoint afin que les utilisateurs puissent masquer des réponses par programmation selon un flux personnalisé qu’ils définissent.

[Aller à l’application ▸](https://pandaflow.io/ "Aller à l’application ▸")

[Pandaflow sur X](https://x.com/pandaflowio "Pandaflow sur X")
