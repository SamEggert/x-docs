<div id="overview">
  ## Aperçu
</div>

Un objet multimédia représente une photo, une vidéo ou un GIF animé. Les objets multimédias sont utilisés par de nombreux points de terminaison de l’API X et peuvent être inclus dans les Tweets, les Messages privés, les profils d’utilisateurs, les créations publicitaires, etc. Chaque objet multimédia peut avoir plusieurs variantes d’affichage ou de lecture, avec différentes résolutions ou formats.

<div id="media-types-size-restrictions">
  ## Types de médias et restrictions de taille
</div>

Restrictions de taille pour le chargement via l’API 

- Image : 5 Mo
- GIF : 15 Mo
- Vidéo : 512 Mo (avec media_category=amplify)

<div id="creation">
  ## Création
</div>

Des objets tels que des Tweets, des Messages privés, des photos de profil, des cartes publicitaires hébergées, etc. peuvent contenir un ou plusieurs objets média. Ces objets de premier niveau sont collectivement appelés entités. L’API de création d’entité correspondante (par ex. POST statuses/update) peut recevoir un ou plusieurs objets média à l’aide d’un media_id unique.

Une entité contenant un ou plusieurs objets média peut être créée en suivant ces étapes :

1. Importez le ou les fichiers média en utilisant soit le téléversement segmenté recommandé (images/GIF/vidéo), soit l’ancien téléversement simple (images uniquement).
2. Récupérez un media_id à l’issue de l’étape 1. Cette étape peut être répétée plusieurs fois avec différents médias si l’entité autorise la transmission de plusieurs paramètres media_id.
3. Créez l’entité en appelant le point de terminaison approprié, en incluant le media_id et les autres paramètres requis. Par exemple, joignez un media_id à un Tweet à l’aide du point de terminaison POST statuses/update.

<div id="retrieving">
  ## Récupération
</div>

Veuillez vous reporter à l’objet Media dans le dictionnaire de données des Tweets.
