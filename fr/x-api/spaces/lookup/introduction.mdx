Les endpoints de recherche par ID renvoient des données que vous pouvez utiliser pour concevoir des expériences affichant des informations sur les Spaces à vos utilisateurs, ou pour obtenir des détails sur un ou plusieurs Spaces. Vous pouvez interroger un ou plusieurs Spaces dans une même requête.

Vous pouvez également rechercher des Spaces par l’ID de leur créateur. Cet endpoint est utile pour récupérer les Spaces d’utilisateurs figurant dans une liste, ou pour déterminer si plusieurs hôtes récurrents sont actuellement en direct ou ont programmé un Space à venir. Jusqu’à 100 utilisateurs peuvent être interrogés en une seule requête.

**Configuration du compte**

Pour accéder à ces endpoints, vous aurez besoin de :

* D’un [compte développeur](https://developer.x.com/en/portal/petition/essential/basic-info) approuvé.
* De vous authentifier à l’aide des clés et jetons d’une [application développeur](/fr/resources/fundamentals/developer-apps) située au sein d’un [projet](/fr/resources/fundamentals/projects).

En savoir plus sur l’accès aux endpoints X API v2 dans notre [guide de démarrage](/fr/x-api/getting-started/getting-access).
