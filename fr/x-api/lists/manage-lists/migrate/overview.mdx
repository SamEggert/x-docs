<div id="comparing-x-apis-lists-endpoints">
  ## Comparaison des endpoints Lists de l’API X
</div>

Les endpoints de gestion des Lists en v2 finiront par remplacer [POST lists/create](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-create), [POST lists/destroy](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-destroy) et [POST lists/update](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-update). Si votre code, vos applications ou vos outils utilisent une version antérieure de ces endpoints et que vous envisagez de migrer vers la X API v2, ce guide est pour vous.

Les tableaux suivants comparent les endpoints List de la version standard v1.1 et de la X API v2 :

**Créer une List**

| Description | Standard v1.1 | X API v2 |
| :--- | :--- | :--- |
| Méthodes HTTP prises en charge | POST | POST |
| Domaine hôte | https://api.x.com | https://api.x.com |
| Chemin de l’endpoint | /1.1/lists/create.json | /2/lists |
| [Authentication](/fr/resources/fundamentals/authentication) | Contexte utilisateur OAuth 1.0a | Contexte utilisateur OAuth 1.0a |
| [Rate limits](/fr/resources/fundamentals/rate-limits) de requêtes par défaut | Aucune | 300 requêtes par 15 min (par utilisateur) |

**Supprimer une List**

| Description | Standard v1.1 | X API v2 |
| :--- | :--- | :--- |
| Méthodes HTTP prises en charge | POST | DELETE |
| Domaine hôte | https://api.x.com | https://api.x.com |
| Chemin de l’endpoint | /1.1/lists/destroy.json | /2/lists/:id |
| [Authentication](/fr/resources/fundamentals/authentication) | Contexte utilisateur OAuth 1.0a | Contexte utilisateur OAuth 1.0a |
| [Rate limits](/fr/resources/fundamentals/rate-limits) de requêtes par défaut | Aucune | 300 requêtes par 15 min (par utilisateur) |

**Mettre à jour une List**

| Description | Standard v1.1 | X API v2 |
| :--- | :--- | :--- |
| Méthodes HTTP prises en charge | POST | PUT |
| Domaine hôte | https://api.x.com | https://api.x.com |
| Chemin de l’endpoint | /1.1/lists/update.json | /2/lists/:id |
| [Authentication](/fr/resources/fundamentals/authentication) | Contexte utilisateur OAuth 1.0a | Contexte utilisateur OAuth 1.0a |
| [Rate limits](/fr/resources/fundamentals/rate-limits) de requêtes par défaut | Aucune | 300 requêtes par 15 min (par utilisateur) |

Pour accéder aux endpoints de la X API v2, vous devez [créer un compte développeur](https://developer.x.com/en/portal/petition/essential/basic-info). Lors de l’authentification, utilisez des clés et des jetons provenant d’une [developer App](/fr/resources/fundamentals/developer-apps) située dans un [Project](/fr/resources/fundamentals/projects).

Pour en savoir plus sur l’accès aux endpoints de la X API v2, consultez notre page [getting started](/fr/x-api/getting-started/getting-access).
