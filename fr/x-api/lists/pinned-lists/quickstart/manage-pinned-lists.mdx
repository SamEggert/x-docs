<div id="steps-to-build-a-manage-pinned-list-request">
  ### Étapes pour créer une requête de gestion des Listes épinglées
</div>

**Première étape : choisir la collection d’endpoints List dans Postman**

Une fois la collection X API v2 chargée dans Postman, accédez au dossier “List”, sélectionnez le dossier “Pinned Lists”, puis choisissez &quot;Pin a List&quot;.
 

**Deuxième étape : spécifier la Liste à épingler**

Les endpoints de gestion des Listes épinglées requièrent deux identifiants : un pour l’utilisateur (l’utilisateur authentifié qui épingle une Liste) et la Liste cible (la Liste à épingler). L’ID de l’utilisateur doit correspondre à l’ID de l’utilisateur qui s’authentifie, ce qui signifie que vous devez transmettre les Access Tokens associés à cet ID lors de l’authentification de votre requête. Dans ce cas, vous pouvez indiquer l’ID de votre propre utilisateur. Vous pouvez trouver votre ID de deux façons :

1. En utilisant l’endpoint [users lookup](/fr/x-api/users/lookup/introduction) par nom d’utilisateur, vous pouvez transmettre un nom d’utilisateur et recevoir le champ id.
2. En consultant votre Access Token, vous constaterez que la partie numérique correspond à votre ID utilisateur.

L’ID de la Liste cible peut être celui de n’importe quelle liste valide. Dans Postman, accédez à l’onglet &quot;Params&quot; et saisissez l’ID utilisateur dans la colonne &quot;Value&quot; de la variable de chemin id. Accédez à l’onglet “Body” et renseignez l’ID de la Liste que vous souhaitez épingler comme valeur du paramètre list&#95;id. Veillez à ne pas inclure d’espaces avant ou après un ID.

|     |     |     |
| :--- | :--- | :--- |
| **Key** | **Value** | **Parameter type** |
| `id` | L’ID utilisateur | path |
| list&#95;id | L’ID de la Liste cible à épingler | body |

Vous devriez maintenant voir une URL similaire à côté du bouton &quot;Send&quot; :

```
      https://api.x.com/2/users/2244994945/pinned_lists
```

Troisième étape : effectuez votre requête et vérifiez la réponse

Une fois tout configuré, cliquez sur le bouton &quot;Send&quot; ; vous recevrez une réponse similaire à l’exemple suivant :

```
{
  "data": {
    "pinned": true
  }
}
```

La Liste cible a été épinglée avec succès si l’objet de réponse contient la valeur true pour la clé pinned. 

Pour désépingler une Liste, sélectionnez la requête &quot;Unpin List&quot; également disponible dans le dossier “Lists” de la collection X API v2 chargée dans Postman. Cet endpoint requiert l’ID de l’utilisateur authentifié et l’ID de la Liste à désépingler. Dans l’onglet “Params”, saisissez l’ID utilisateur comme valeur de la colonne id et l’ID de la Liste à désépingler comme valeur de la colonne list&#95;id. 

En cas de suppression réussie, vous recevrez une réponse similaire à l’exemple suivant :

```
{
  "data": {
    "pinned": false
  }
}
```
