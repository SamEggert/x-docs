  ## Comparaison des endpoints de recherche de listes de l’API X
</div>

Le groupe d’endpoints de recherche de listes v2 remplacera les endpoints standard v1.1 [GET lists/show](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-show) et [GET lists/ownership](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-ownerships). Si vous avez du code, des applications ou des outils qui utilisent l’une de ces versions des endpoints de recherche de listes et que vous envisagez de migrer vers le nouvel endpoint X API v2, alors cette série de guides est faite pour vous.

Les tableaux suivants comparent les endpoints de listes standard v1.1 et X API v2 :

**Recherche de liste par ID**

|     |     |     |
| :--- | :--- | :--- |
| Description | Standard v1.1 | X API v2 |
| Méthodes HTTP prises en charge | `GET` | `GET` |
| Domaine hôte | `https://api.x.com` | `https://api.x.com` |
| Chemin de l’endpoint | `/1.1/lists/show.json` | `/2/lists/:id` |
| [Authentication](/fr/resources/fundamentals/authentication) | OAuth 1.0a User Context<br /><br />App only | OAuth 1.0a User Context<br /><br />OAuth 2.0 Authorization Code with PKCE<br /><br />App only |
| [Rate limits](/fr/resources/fundamentals/rate-limits) de requêtes par défaut | 75 requêtes par 15 min avec OAuth 1.0a<br /><br />75 requêtes par 15 min avec OAuth 2.0<br /><br />75 requêtes par 15 min avec App only | 75 requêtes par 15 min avec OAuth 1.0a<br /><br />75 requêtes par 15 min avec OAuth 2.0<br /><br />75 requêtes par 15 min avec App only |

**Recherche des listes détenues par l’utilisateur**

|     |     |     |
| :--- | :--- | :--- |
| Description | Standard v1.1 | X API v2 |
| Méthodes HTTP prises en charge | `GET` | `GET` |
| Domaine hôte | `https://api.x.com` | `https://api.x.com` |
| Chemin de l’endpoint | `/1.1/lists/ownerships.json` | `/2/users/:id/owned_lists` |
| [Authentication](/fr/resources/fundamentals/authentication) | OAuth 1.0a User Context<br /><br />App only | OAuth 1.0a User Context<br /><br />OAuth 2.0 Authorization Code avec PKCE<br /><br />App only |
| [Rate limits](/fr/resources/fundamentals/rate-limits) de requêtes par défaut | 15 requêtes par 15 min avec OAuth 1.0a<br /><br />15 requêtes par 15 min avec App only | 15 requêtes par 15 min avec OAuth 1.0a<br /><br />15 requêtes par 15 min avec OAuth 2.0<br /><br />15 requêtes par 15 min avec App only |

Pour accéder aux endpoints X API v2, vous devez [vous inscrire pour obtenir un compte développeur](https://developer.x.com/en/portal/petition/essential/basic-info). Lors de l’authentification, vous devez utiliser des clés et des jetons provenant d’une [developer App](/fr/resources/fundamentals/developer-apps) qui se trouve dans un [Project](/fr/resources/fundamentals/projects).

Pour en savoir plus sur l’accès aux endpoints X API v2, consultez notre page [getting started](/fr/x-api/getting-started/getting-access).
